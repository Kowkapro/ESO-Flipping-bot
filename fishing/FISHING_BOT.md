# Fishing Bot — Документация и план разработки

> **Статус:** Phase A-D написаны и частично протестированы. Модель v3 в работе.
> **Основной файл:** `fishing/main.py`
> **Альтернативный:** `fishing/yolo_fisher.py` (модульный, класс-based)

---

## Что делает бот

Автоматически рыбачит в ESO: ходит между рыболовными лунками, ловит рыбу, переходит к следующей лунке. Работает полностью визуально — через YOLO нейросеть "смотрит" на экран игры.

### Полный цикл (одна итерация)

```
1. Открыть карту (M)
2. Приблизить карту (10 кликов на "+")
3. YOLO-скан: найти синие крючки (blue_hook) на карте
4. Отфильтровать уже посещённые лунки и иконку игрока (PLAYER_ICON_RADIUS=60px)
5. Выбрать ближайшую к игроку -> снять старую метку (F) -> поставить новую метку (F) -> закрыть карту

6. Повернуть камеру к метке на компасе (Phase B)
7. Бежать спринтом к метке, корректируя курс по компасу (Phase C)
8. Определить прибытие (маркер исчез / прыгнул / interaction_prompt)

9. Осмотреться (4x90 поворот), найти лунку (interaction_prompt / bubbles)
10. Рыбачить: заброс (E) -> ждать крючок -> подсечка (E) -> лут (R) -> повтор
11. Лунка иссякла (2 пустых заброса) -> перейти к шагу 1
```

## Архитектура (4 фазы)

### Phase A — Выбор лунки (карта)
- Открываем карту, зумим на максимум
- YOLO находит все `blue_hook` иконки
- Дедупликация (лунки ближе 30px = одна и та же)
- Фильтрация уже посещённых (хранятся как dx,dy от центра экрана)
- Кликаем на ближайшую, ставим метку (F), закрываем карту

**Ключевые параметры:**
- `MAP_ZOOM_CLICKS = 10` — количество кликов зума
- `HOOK_MIN_CONF = 0.3` — минимальная уверенность YOLO
- `HOOK_DEDUP_DIST = 30px` — порог дедупликации
- `VISITED_DEDUP_DIST = 40px` — порог "уже посещена"

### Phase B — Поворот к метке
- YOLO ищет `compass_marker` на полоске компаса
- Рассчитывает смещение от центра экрана
- Плавно поворачивает камеру (human_mouse_arc, S-curve, 500Hz)
- До 20 попыток выровняться

**Ключевые параметры:**
- `MARKER_MIN_CONF = 0.3`
- `SCREEN_TO_MOUSE = 2.42` — конвертация пикселей экрана в пиксели мыши
- `DEAD_ZONE_FRAC = 0.02` — "мёртвая зона" (считается выровненным)
- `STEER_DAMPING = 0.9` — демпфирование (чтобы не перелетать)

### Phase C — Бег к метке
- Спринт (W + Shift) с коррекцией курса по компасу
- Каждые 150мс: YOLO-скан -> проверка маркера -> коррекция

**Определение прибытия (по приоритету):**
1. **OCR-проверка interaction_prompt** — YOLO видит промпт -> EasyOCR читает текст -> если содержит "рыбалк"/"ловл" -> FISHING (иначе игнор)
2. Маркер прыгнул >= 250px за 1 кадр -> ARRIVED (прошли мимо метки)
3. **Кружение** — маркер меняет сторону 4+ раз за 8 кадров -> ARRIVED (крутимся вокруг точки)
4. Маркер потерян 4+ кадров подряд -> ARRIVED (подошли вплотную)
5. Таймаут 60 сек -> TIMEOUT

> **OCR:** YOLO не читает текст — "Место рыбалки" и "Дорожное святилище" визуально одинаковые (`interaction_prompt`). EasyOCR (ru+en) распознаёт текст кропа и фильтрует по ключевым словам.

**Антизастревание:**
- Каждые 2 сек проверяет: сместился ли маркер хотя бы на 15px
- Если нет -> прыжки (2-3 раза)
- Детекция кружения: маркер прыгает слева-направо = бот рядом с целью

**Ключевые параметры:**
- `RUN_STEER_DAMPING = 0.7`
- `RUN_MAX_DURATION = 60.0s`
- `MARKER_JUMP_THRESHOLD = 250px`
- `MARKER_LOST_THRESHOLD = 4` кадров (было 8)
- `CIRCLING_FLIP_THRESHOLD = 4` смен знака за 8 кадров
- `STUCK_CHECK_INTERVAL = 2.0s`
- `FISHING_KEYWORDS = ["рыбалк", "ловл", "fishing", "рыбн"]`

### Phase D — Рыбалка
- Заброс (E) -> ждать белый крючок (Votan's Fisherman) -> подсечка (E) -> лут (R)
- 2 пустых заброса подряд = лунка иссякла
- Все задержки рандомизированы (имитация человека)

**Ключевые параметры:**
- `MAX_WAIT_FOR_HOOK = 45s` — макс ожидание клёва
- `MAX_FAILED_CASTS = 2` — пустых забросов до "иссякла"
- `HOOK_WHITE_THRESHOLD = 220` — яркость пикселя для определения крючка
- `HOOK_WHITE_RATIO = 0.08` — доля белых пикселей

## YOLO Модель

### Модель v2 (устаревшая, слабая)
- **Архитектура:** YOLOv8s (small)
- **Путь:** `fishing/training/runs/eso_fishing/weights/best.pt`
- **Обучение:** 231 изображений, 468 аннотаций, mAP50=0.879
- **8 классов:** blue_hook, bubbles, compass_marker, enemy, interaction_prompt, hp_bar, xp_popup, destination_text
- **Проблемы:** не различает waypoint/quest маркеры, fishing/npc промпты, bubbles слабый (mAP50=0.359)

### Модель v3 (текущая, обучается)
- **Архитектура:** YOLO11l (large) — лучшая точность
- **imgsz:** 1280 — лучше видит мелкие объекты
- **batch:** 2 (для 8GB VRAM с amp=True)
- **Скрипт обучения:** `fishing/training/train.py`
- **Путь модели:** `fishing/training/runs/eso_fishing_v3/weights/best.pt`
- **Датасет v3:** 403 изображения, 2772 аннотации, 10 классов
- **Baseline на v2 данных:** mAP50=0.910 (уже +3.5% vs v2)

### Классы модели v3 (10 штук)

| ID | Класс | Описание | Авто-разметка |
|----|-------|----------|---------------|
| 0 | fishing_hook | Синий крючок HarvestMap на карте | YOLO-World |
| 1 | player_icon | Синий треугольник игрока на карте | YOLO-World |
| 2 | waypoint_pin | Ромб метки на карте | Ручная |
| 3 | waypoint_marker | Маркер нашей метки на компасе | Ручная |
| 4 | quest_marker | Маркер квеста/POI на компасе | Ручная |
| 5 | fishing_prompt | "[E] Место рыбалки..." | Claude Vision |
| 6 | npc_prompt | "[E] Поговорить" и т.д. | Claude Vision |
| 7 | bubbles | Пузыри рыбной лунки | Claude Vision |
| 8 | enemy | Враждебный NPC | Claude Vision |
| 9 | hp_bar | Полоска HP (своя или врага) | Claude Vision |

### Авто-разметка скриншотов

Подробная документация: `fishing/training/AUTO_ANNOTATOR.md`

**Гибридный подход** (вместо ручной разметки в CVAT):
- **YOLO-World** (`test_yolo_world.py`) — zero-shot detection для мелких иконок на карте (крючки ~20px). Recall ~50%, но точная локализация.
- **Claude Vision** (`auto_annotate.py`) — API через polza.ai для крупных объектов (промпты, пузыри, враги, HP бары). Хорошо понимает контекст и UI.
- **Ручная разметка в CVAT** — для компасных маркеров и финальной ревизии.

**Результаты тестирования (06.03.26):**
| Инструмент | Мелкие объекты (карта) | Крупные объекты (промпты) | Компас |
|-----------|----------------------|--------------------------|--------|
| Claude Vision | Плохо (галлюцинации) | Хорошо | Не тестирован |
| YOLO-World small | **885 hooks на 262 картах (~50% recall)** | Не находит | Не находит |
| YOLO-World medium/large | **0 детекций** (хуже small!) | Не находит | Не находит |
| CVAT (ручная) | Идеально | Идеально | Идеально |

**YOLO-World:** только `yolov8s-worldv2` + промпт `"blue pin icon"` работает. Все остальные комбинации модель/промпт дают 0.

## Управление ботом

- **F5** — Старт (сначала переключиться на ESO!)
- **F6** — Стоп (отпускает все клавиши)

### Запуск
```bash
python fishing/main.py
```

### Предварительные требования
- ESO в оконном полноэкранном режиме (1920x1080)
- Аддон Votan's Fisherman включён (показывает белый крючок)
- Аддон HarvestMap включён (показывает синие крючки на карте)
- YOLO модель v3 обучена (`fishing/training/runs/eso_fishing_v3/weights/best.pt`)
- Запуск от администратора (для keyboard hooks)
